# My Awesome Project

This is a full description of my project. It is a linter that helps improve the quality of documentation in open-source projects, making them more accessible to new contributors.

## Installation
To install, you just need Python 3. No other libraries are required.

## Usage
Run the script from your terminal: `python readme_check.py <filename>`
